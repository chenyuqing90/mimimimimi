Azure VM

2023/3/14

以下是在 Azure 上建立 Windows 虚拟机 (VM) 并将其用作服务器的一些步骤：

  1. 登录到 Azure 门户 （https://portal.azure.com/）。
  2. 选择“创建资源”，然后选择“计算机”，接着选择“虚拟机”。
  3. 在“虚拟机”页上，选择 Windows 操作系统。
  4. 根据需要选择 VM 的大小、存储选项等。
  5. 配置网络设置，包括虚拟网络、子网、公共 IP 等。
  6. 配置安全设置，如 NSG 规则和防火墙设置。
  7. 配置管理选项，如管理员用户名和密码。
  8. 完成配置后，选择“创建”。
  9. Azure 将开始创建 VM。在创建过程中，你可以监视进度并查看 VM 的状态。
  10. 创建完成后，你可以通过远程桌面连接到 VM，并将其用作服务器。
